# 챗쪽이팀 Project 

## ✯ 성격 기반 AI 챗봇 "챗쪽이"
* 본인의 성격을 기반으로 한 가상의 "나의 아이"와 대화하기
* 나의 아이의 성향을 테스트해볼 수 있는 간단한 TEST 검사 페이지 제작

## ☑ 사용한 데이터
* [KoGPT2](https://github.com/SKT-AI/KoGPT2)
* 필요한 성격별 데이터 수기 입력.
